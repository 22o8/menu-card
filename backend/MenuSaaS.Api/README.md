# Menu Book SaaS Backend

Back-end starter باستخدام ASP.NET Core 8 Web API.

## الموجود حاليًا
- MenuBooks endpoints
- Themes endpoint
- Demo in-memory store
- Swagger

## التشغيل
```bash
dotnet restore
dotnet run --project MenuSaaS.Api/MenuSaaS.Api.csproj
```

## المراحل القادمة المقترحة
- JWT Authentication
- PostgreSQL + EF Core
- S3 / Cloudflare R2 uploads
- Publish links + QR codes
- Analytics
- Theme CRUD

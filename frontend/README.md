# Menu Book SaaS Frontend

واجهة Nuxt 3 تحتوي على:
- Landing page
- Admin Dashboard
- Menu Builder
- Theme presets
- FlipBook Viewer باستخدام page-flip

## التشغيل

```bash
npm install
npm run dev
```
